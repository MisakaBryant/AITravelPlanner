# AITravelPlanner 前端开发说明

本前端基于 React + Vite + TypeScript + Ant Design + 高德地图。

## 快速开始

1. 进入 frontend 目录，安装依赖：
   ```powershell
   npm install
   ```
2. 复制 `../config/.env.example` 为 `.env`，并填写前端所需 API Key（如高德地图、语音识别等）。
3. 启动开发服务：
   ```powershell
   npm run dev
   ```
4. 访问 http://localhost:5173 查看页面。

> 详细页面与组件开发将在后续补充。
